# Adopt zhiyun-style anti-AI-slop rules for the WOA Web

The WOA SaaS Web entrypoint will adopt a project-specific version of the anti-AI-slop frontend discipline from `zhiyun-ai-dashboard`: Astryx semantic tokens, form/table/definition-list-first layouts, restrained motion, specific operational copy, and explicit bans on common AI-generated SaaS visual tropes. These rules are captured in `docs/design/anti-ai-slop-rules.md` and are part of Web acceptance.
